# 퍼스트 퀸 IV 재현 목업 — 바르시아 전기 (원작 고증판)

呉ソフトウェア工房의 1994년작 **「ファーストクイーンIV ~バルシア戦記~」**(퍼스트 퀸 IV)를
웹(바닐라 JS + 캔버스)으로 재현한 팬 오마주입니다. 실제 등장인물·성(지역)·스토리 전개를
조사해 반영했습니다. **HTML + CSS + Vanilla JavaScript** — 프레임워크·빌드·외부 에셋 없음.

> 대사 텍스트는 원작 줄거리를 따라 새로 쓴 창작이며, 그래픽은 전부 캔버스 절차 드로잉입니다.
> 일부 동료 합류 시점·수치는 웹 목업에 맞게 근사했습니다. 개인 소장용 팬 프로젝트입니다.

## 실행 방법

`index.html`을 브라우저로 열면 바로 실행됩니다. 로컬 서버 사용 시(권장):

```bash
python3 -m http.server 8000   # http://localhost:8000
```

## 조작법

| 조작 | 키 |
| --- | --- |
| 이동 | 방향키 / WASD (모바일: 좌측 조이스틱) |
| 공격 | 적과 부딪치면 자동 공격 |
| 특수능력·소환 | `스페이스` |
| 조작 캐릭터 교대 / 군단 전환 | `Tab`·초상 클릭 / `Q` |
| 부대 명령 | `1` 자유 / `2` 집합 / `3` 돌격 |
| 지원군 호출 | `R` (출진하지 않은 예비대를 전장에 투입, 전투당 2회) |
| 아이템 | `4` 파괴의 돌(광역 대미지) / `5` 회복약 |
| 퇴각 | `ESC` |

## 원작에서 재현한 시스템

- **고챠캐릭(ごちゃキャラ)** — 수십 명이 한 화면에서 실시간 난전. 조작하지 않는 부대원은 전원 AI 교전
- **AT/DF + AR/DR** 능력치와 **피로도(FT)** — 피로하면 방어가 급락해 고레벨도 순식간에 죽는 원작 감각
- **리더십제 부대 편성** — 부대장의 리더십이 이끌 수 있는 인원을 결정. **아레스는 리더십이 낮아
  부대장 전용**(원작에서 첫 전투가 어려운 이유), **라딘칼(리더십 12)**이 주력 부대장
- **2개 군단 동시 운용** — `Q`로 군단 전환, 군단별 부대장 추종
- **소환수** — 아레스 「사라만다」(모로시아 해방 후) · 카라 「용」 · 바르톰 「실프」(리스레이 후) ·
  니먼 「운디네」(에드윈 후, 치유의 물결) — 실프/운디네는 **비행**으로 물 위를 건넌다
- **지원군 호출 / 파괴의 돌** — 원작의 전술 커맨드·결전 아이템
- **네임드 영구 사망**, 적 증원, 전략(턴제 지역 점령)+전투(실시간) 이중 구조
- **인물 도감** — 만난 아군·적장의 원작 설정 열람

## 스토리 (원작 전개)

가이아 대륙의 싸움(전작)에서 20년 — 그리고 「퍼스트 퀸」 1편 시대의 아득한 과거(프리퀄).
로그리스 대륙의 마법대국 **바르시아**의 현왕 **제넬루**는 마도사 **자닐**을 참모로 맞은 뒤 돌변,
**인형병사**로 군비를 불려 대륙을 침탈한다. 카리온의 젊은 왕 **아레스**는 제넬루 암살에 실패해
투옥되지만 **의문의 마법사**에게 구출되어 귀환하고, 가이아의 영웅 **아이라 여왕**이 보낸
지원군과 함께 반격을 시작한다. 최종보스는 자닐의 정체인 시리즈의 숙적 **가로아**.

## 아군 (원작 등장인물)

| 인물 | 원작 설정 반영 |
| --- | --- |
| 아레스 | 주인공, 카리온의 왕. 검+마법. 리더십 낮음(부대장 전용). 사라만다 소환 |
| 애레인 | 아레스의 연인. 닌자 부대를 이끌고 바르시아를 수색하다 합류 |
| 카이 | 동마시아성 왕궁에 홀로 낙오해 있던 애레인 부대의 닌자 |
| 라딘칼 | 아이라 여왕의 지원군 기사단장. 리더십 12 — 초반 주력 부대장 |
| 얀후레트 | 아이라 여왕의 아들. 능력치는 낮지만 「함성」으로 아군 피로 회복 |
| 카라 | 소환술사. 사라만다에 버금가는 「용」을 소환 |
| 니먼 | 물의 요정왕. 벨더성 해방 후 합류. 방어율·HP 최상급. 운디네 소환 |
| 바르톰 | 바람의 정령왕. 썬리스의 탑에서 갑옷기사가 된 그를 해방하면 합류. 실프 소환 |
| 크로비 | 동마시아의 왕자. 몰아붙이면 항복하고 중갑보병과 함께 합류 |

## 진행 (원작 공략 순서)

1. **바르시아 국경 요새** — 인형병사 선봉대 (벨더 직행처럼 보여도 반드시 요새 경유 — 원작 재현)
2. **벨더성** — 불의 장군 **오그**(거대한 용, **약점은 꼬리 — 등 뒤에서 2배 피해**)
3. **썬리스의 탑** *(우회 가능)* — 최상층의 갑옷기사 = 사로잡힌 **바르톰**
4. **리스레이성** — 흙의 장군 **맥가이어**(강철 골렘: 지진·석화의 눈빛·골렘 소환, 부하는 마법에 약함)
5. **링커 마을** — **자경단**과의 공동 전선. 모로시아로 가는 길이 열린다
6. **모로시아성** — 물의 장군 **몰드레드**(해자를 헤엄치는 물뱀, **바람의 정령에 약함**) → 사라만다 해금
7. **동마시아성** *(우회 가능)* — **크로비 왕자** 항복 이벤트, 카이 구출
8. **에드윈성** — **뱀파이어**(흡혈)를 쓰러뜨려 **운디네 해방** → 바람의 장군 **스리후트**
9. **바르시아 왕성** — 조종당하는 **제넬루 왕** → 정체를 드러내는 **자닐 = 가로아** 2단계 최종전

## 파일 구성

```
index.html          게임 페이지(타이틀/전략맵/출진/전투/결과/도감)
css/                common.css(공용) + firstqueen.css(게임)
js/fq_data.js       인물·병종·사천왕·소환수 / 성 9곳 맵 생성기 / 스토리 텍스트
js/firstqueen.js    전략맵·리더십 편성·실시간 전투 엔진(AI/소환/비행/보스 기믹)·저장
```

진행은 `localStorage`에 자동 저장됩니다(이어하기 지원). 모바일 터치 조작 지원.

## 원작과 다른 점(의도적 간소화)

- 패배/퇴각 시 부대·아이템이 출진 전 상태로 복원(원작이라면 세이브 로드)
- 성 내부 탐색·상점·적 턴의 역침공은 미구현, 부대는 최대 2개 군단으로 단순화
- 일부 합류 시점(카라 등)과 수치는 근사치
