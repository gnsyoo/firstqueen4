# 퍼스트 퀸 IV 재현 목업 — 바르시아 전기

呉ソフトウェア工房의 1994년작 **「ファーストクイーンIV ~バルシア戦記~」**(퍼스트 퀸 IV)의
맵 구성과 플레이 감각을 웹으로 재현한 오마주 목업입니다.
**HTML + CSS + Vanilla JavaScript** — 프레임워크·빌드 도구·외부 에셋 없이 동작합니다.

## 실행 방법

`index.html`을 브라우저로 여는 것만으로 실행됩니다. 로컬 서버 사용 시(권장):

```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

모바일에서는 PC와 같은 와이파이에서 `http://<PC-IP>:8000` 으로 접속하세요.
(가상 조이스틱·터치 버튼 지원)

## 원작에서 재현한 요소

| 원작 시스템 | 재현 방식 |
| --- | --- |
| **고챠캐릭(ごちゃキャラ)** — 수십 명이 한 화면에서 실시간 난전 | 최대 아군 36명(18명 군단 ×2) vs 적 수십 명. 조작하지 않는 부대원은 전원 AI로 스스로 이동·교전 |
| 리더 1명 직접 조작 + 캐릭터 교대 | 아무 부대원이나 교대 조작 가능 |
| 능력치 **AT/DF(공격·방어력) + AR/DR(공격율·방어율)** | 명중 판정은 AR vs DR, 피해는 AT vs DF — 원작의 곱연산식 재현 |
| **피로도(FT)** — 높으면 방어가 급락 | 공격/피격으로 상승, 대기·휴식으로 회복. FT가 높은 유닛은 순식간에 죽는 원작 감각 그대로 |
| 네임드 캐릭터 **영구 사망** | 영웅(★)이 전사하면 그 세이브에서 영원히 이탈 |
| 전략(턴제 지역 이동) + 전투(실시간) 이중 구조 | 로그리스 대륙 전략맵에서 인접 지역으로 진군 → 실시간 전투 |
| 부대 편성(1부대 18명, 동시 2부대) | 출진 화면에서 최대 36명(제1·제2군단) 선택 배치 |
| 적 지원군 / `스페이스` 특수능력 / 부대 명령 | 전투 중 조건부 적 증원 · 캐릭터 고유기(화염참·수리검 난무 등) · 자유/집합/돌격 |

## 스토리(원작 기반)

- 무대: **로그리스 대륙**. 마법대국 **바르시아**의 현왕 **제넬**이 타 대륙에서 온 마도사
  **자닐**(정체는 시리즈의 숙적 **가로아**)을 참모로 맞은 뒤 돌변, **인형병사**로 군비를 늘려
  정복 전쟁을 일으킨다.
- 주인공: 남부 소국 카리온의 왕자 **아레스**(불의 정령왕의 힘을 지님). 연인 **애레인**은
  닌자 부대를 이끌고 합류한다.
- 적장: 바르시아 사장군 — **오그**(물)·**스리프트**(바람)·**가이아**(땅)·**모드레드**(불),
  그리고 **제넬 왕** → 정체를 드러내는 **자닐(가로아)** 2단계 최종전.

## 지역(전투 맵) 구성

1. **남부 국경 평원** — 초원+숲, 남북 가도. 튜토리얼급 전투
2. **로그리스 대교** — 대륙을 가르는 강. 좁은 다리 위 공방전 (보스: 오그)
3. **서부 대삼림** — 거목의 바다, 좁은 숲길과 복병 (보스: 스리프트)
4. **동부 바위고개** — S자 협로, 중장인형의 길목 방어 (보스: 가이아)
5. **중앙 요새 리간** — 성벽+성문 공성전 (보스: 모드레드)
6. **북부 설원** — 인형 대군과의 총력전
7. **바르시아 왕도** — 왕성 진입, 제넬 왕 → 자닐 연전

서부/동부는 **분기 루트**로, 한쪽만 뚫어도 중앙 요새로 진군할 수 있습니다.

## 파일 구성

```
index.html          게임 페이지(타이틀/전략맵/출진/전투/결과)
css/common.css      공용 기본 스타일(버튼·모달 등)
css/firstqueen.css  게임 전용 스타일
js/fq_data.js       병종·영웅·보스 정의, 지역 그래프, 결정적 맵 생성기, 스토리 텍스트
js/firstqueen.js    전략맵·출진 편성·실시간 전투 엔진(AI/충돌/투사체/미니맵/HUD)·저장
```

- 진행 상황은 브라우저 `localStorage`에 자동 저장됩니다(이어하기 지원).
- 유닛/타일 아트는 전부 캔버스 절차 드로잉 — 외부 이미지·라이브러리 없음.
